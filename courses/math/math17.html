📘 Application of Differentiation

JAMB Mathematics • rate of change • chain rule • stationary points • maxima and minima • optimization

📋 Chapter Overview

This chapter covers one of the most important parts of introductory calculus in JAMB: Applications of Differentiation.

The syllabus focus is mainly on:

  • Rate of change
  • Maxima and minima
  • Rate of change tells how fast one quantity changes compared with another
  • Stationary points happen where the gradient is zero
  • Second derivative helps you decide whether the point is a maximum or a minimum
  • This topic is high-yield in JAMB and often gives 2–3 questions
🧠 Big idea

Differentiation is not only about finding derivatives. In this chapter, the derivative is used to solve real problems involving speed, area, volume, turning points, and optimization.

🔑 Essential Formulas

Application Formula / Rule Explanation
Rate of Change dy/dx or ds/dt Instantaneous rate of change
Chain Rule dy/dt = (dy/dx)(dx/dt) Connects related rates
Stationary Points Set dy/dx = 0 Points where gradient is zero
Second Derivative d²y/dx² Determines the nature of turning points
Maximum Point dy/dx = 0 and d²y/dx² < 0 Curve is concave downward
Minimum Point dy/dx = 0 and d²y/dx² > 0 Curve is concave upward
Point of Inflection d²y/dx² = 0 Concavity changes

1️⃣ Rate of Change

Concept and Applications

Rate of change measures how fast one quantity varies with another.

Key rates
ds/dt = rate of change of distance or displacement
dA/dt = rate of change of area
dV/dt = rate of change of volume

Important idea: If one quantity depends on another quantity that is changing with time, use the chain rule.

Rate of change as slope
The derivative at a point is the slope of the tangent at that point.
Example 1: Square Side Increasing

The side of a square increases at 2 cm/s. Find the rate of increase of its area when the side is 5 cm.

Let side = s, area = A

Given ds/dt = 2

A = s²

Differentiate with respect to t:

dA/dt = 2s · ds/dt

Substitute s = 5 and ds/dt = 2:

dA/dt = 2(5)(2) = 20 cm²/s

Answer: 20 cm²/s
Example 2: Spherical Balloon

A spherical balloon is inflated so that its radius increases at 0.5 cm/s. Find the rate of increase of volume when r = 3 cm. Take π = 22/7.

V = (4/3)πr³

Given dr/dt = 0.5

Differentiate with respect to t:

dV/dt = 4πr² · dr/dt

When r = 3:

dV/dt = 4 × 22/7 × 9 × 0.5

= 396/7 ≈ 56.57 cm³/s

Answer: 396/7 cm³/s or about 56.57 cm³/s

2️⃣ Maxima and Minima

Optimization Problems

  • Differentiate the function
  • Set the first derivative equal to zero
  • Use the second derivative to test the nature
  • Substitute back to get the point or value required
Step-by-step procedure
1. Find dy/dx
2. Solve dy/dx = 0
3. Find d²y/dx²
4. If d²y/dx² > 0 → minimum, if d²y/dx² < 0 → maximum
Turning points on a curve
At a maximum the curve bends downward, and at a minimum it bends upward.
Example 3: Maximum and Minimum Points

Find the maximum and minimum values of y = x³ - 6x² + 9x + 2.

Step 1: Differentiate

dy/dx = 3x² - 12x + 9

Step 2: Set to zero

3x² - 12x + 9 = 0

x² - 4x + 3 = 0

(x - 1)(x - 3) = 0

x = 1 or x = 3

Step 3: Find second derivative

d²y/dx² = 6x - 12

Step 4: Test the points

At x = 1: 6(1) - 12 = -6 < 0 → Maximum

y = 1 - 6 + 9 + 2 = 6

So maximum point is (1, 6)

At x = 3: 6(3) - 12 = 6 > 0 → Minimum

y = 27 - 54 + 27 + 2 = 2

So minimum point is (3, 2)

Answer: Maximum at (1, 6), Minimum at (3, 2)
Example 4: Farmer’s Rectangular Plot

A farmer has 100 m of fencing for a rectangular plot. Find the dimensions for maximum area.

Let length = x, width = y

Perimeter: 2x + 2y = 100

x + y = 50

y = 50 - x

Area:

A = xy = x(50 - x) = 50x - x²

Differentiate:

dA/dx = 50 - 2x

Set to zero:

50 - 2x = 0 → x = 25

Second derivative:

d²A/dx² = -2 < 0 → Maximum

Then y = 50 - 25 = 25

Maximum area = 25 × 25 = 625 m²

Answer: 25 m by 25 m, maximum area = 625 m²
Example 5: Turning Point of a Quadratic

Find the coordinates of the turning point on y = 2x² - 8x + 5 and determine its nature.

dy/dx = 4x - 8

Set to zero:

4x - 8 = 0 → x = 2

Find y:

y = 2(2)² - 8(2) + 5 = 8 - 16 + 5 = -3

Second derivative:

d²y/dx² = 4 > 0

Answer: Turning point = (2, -3), minimum

📝 JAMB Practice Questions

Objective Questions

Q1: Maximum Point

If y = x³ - 3x² + 4, find the value of x at which y is maximum.

Options: A. 0    B. 1    C. 2    D. 3

dy/dx = 3x² - 6x = 3x(x - 2)

So x = 0 or x = 2

d²y/dx² = 6x - 6

At x = 0: -6 < 0 → maximum

At x = 2: 6 > 0 → minimum

Answer: (A) 0
Q2: Circle Area Rate

If the radius of a circle increases at 0.1 cm/s, what is the rate of increase of area when r = 5 cm?

Options: A. π    B. 2π    C. 5π    D. 10π cm²/s

A = πr²

dA/dt = 2πr · dr/dt

= 2π(5)(0.1) = π

Answer: (A) π cm²/s

🎯 Key Points for JAMB

Important reminders

  • Simplify first whenever possible before differentiating
  • The chain rule is very important for related-rates questions
  • Do not forget units in rate-of-change answers
  • Always test each stationary point because a curve may have both maximum and minimum points
  • The second derivative test is the fastest way to classify turning points
  • Read the question carefully: “rate of increase” and “rate of decrease” are not the same wording

Quick reference: d²y/dx² > 0 means minimum, d²y/dx² < 0 means maximum, and d²y/dx² = 0 may indicate a point of inflection.

✅ Pre-Exam Checklist

🧠 Checklist

Know how to identify maximum points, minimum points, and points of inflection. Practise optimization problems involving fencing, boxes, areas, and volumes. Also remember to include units in final answers.
