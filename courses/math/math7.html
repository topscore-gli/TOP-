📋 Introduction to Chapter 7

This chapter focuses on linear and quadratic inequalities, both analytically and graphically. You will learn how to solve inequalities, represent answers on number lines, and interpret shaded solution regions on graphs.

In JAMB, inequality questions are usually easy marks if you remember the sign-flip rule and know how to handle quadratic intervals correctly.

  • Linear inequalities are solved like equations, except for the sign-flip rule
  • Quadratic inequalities usually require factorization and interval testing
  • Strict inequalities use open circles and parentheses
  • Inclusive inequalities use closed circles and square brackets
🧠 Big idea

An inequality does not usually give one answer. It gives a range of values, so think in terms of intervals and regions, not just one number.

1️⃣ Linear Inequalities

Understanding Inequality Symbols

Symbol Meaning Example
< Less than x < 5
> Greater than x > 3
≤ Less than or equal to x ≤ 4
≥ Greater than or equal to x ≥ 2

Key rule

  • The inequality sign flips whenever you multiply or divide both sides by a negative number

Solving Linear Inequalities

Example 1: Solve 3x + 7 < 16

Step 1: Subtract 7 from both sides

3x < 9

Step 2: Divide by 3

x < 3

Answer: x < 3 or (-∞, 3)
Example 2: Solve 5 - 2x ≥ 11

Step 1: Subtract 5

-2x ≥ 6

Step 2: Divide by -2 and flip the sign

x ≤ -3

Answer: x ≤ -3 or (-∞, -3]
Example 3: Solve -3 ≤ 2x + 5 < 13

Step 1: Subtract 5 from all parts

-8 ≤ 2x < 8

Step 2: Divide all parts by 2

-4 ≤ x < 4

Answer: -4 ≤ x < 4 or [-4, 4)

Graphical Representation on Number Line

  • < or > uses an open circle
  • ≤ or ≥ uses a closed circle
  • Shade or draw the line in the direction of the valid values
Number line idea
Open circles are for strict inequalities. Closed circles are for inclusive inequalities.

Linear Inequalities with Fractions

Example 4: Solve x/3 + x/2 > 5

Step 1: Find the LCM of 3 and 2

LCM = 6

Step 2: Multiply every term by 6

6(x/3) + 6(x/2) > 6(5)

2x + 3x > 30

5x > 30

x > 6

Answer: x > 6

2️⃣ Quadratic Inequalities

JAMB Scope

In JAMB, quadratic inequalities usually factorize neatly and have whole-number or simple rational roots.

5-step method
1. Bring everything to one side so the right-hand side becomes 0
2. Factorize the quadratic expression
3. Find the critical values or roots
4. Test intervals or sketch the parabola
5. Choose the correct interval based on the inequality sign
Example 5: Solve x² - 5x + 6 > 0

Step 1: Factorize

(x - 2)(x - 3) > 0

Step 2: Identify critical values

x = 2 and x = 3

Step 3: Test intervals

For x < 2, test x = 0 gives positive

For 2 < x < 3, test x = 2.5 gives negative

For x > 3, test x = 4 gives positive

Step 4: Choose where the expression is positive

x < 2 or x > 3

Answer: (-∞, 2) ∪ (3, ∞)
Example 6: Solve x² + 3x - 10 ≤ 0

Step 1: Factorize

(x + 5)(x - 2) ≤ 0

Step 2: Critical values are -5 and 2

Step 3: Since the parabola opens upward and we want ≤ 0, take the middle interval

-5 ≤ x ≤ 2

Answer: [-5, 2]
Example 7: Solve 2x² < 5x + 3

Step 1: Rearrange

2x² - 5x - 3 < 0

Step 2: Factorize

(2x + 1)(x - 3) < 0

Step 3: Critical values are x = -1/2 and x = 3

Step 4: The expression is negative between the roots

-1/2 < x < 3

Answer: (-1/2, 3)
Example 8: Solve x² - 6x + 9 > 0

Step 1: Factorize

(x - 3)² > 0

Step 2: A square is always non-negative

It equals 0 only when x = 3

Step 3: So the expression is positive for all real x except 3

Answer: x ≠ 3 or (-∞, 3) ∪ (3, ∞)
Quadratic sign idea
For an upward-opening quadratic, values outside the roots are positive, while values between the roots are negative.

Example 9: Graph the inequality 2x + 3y ≤ 6

Step 1: Draw the boundary line 2x + 3y = 6

When x = 0, y = 2 → point (0,2)

When y = 0, x = 3 → point (3,0)

Step 2: Use a solid line because of ≤

Step 3: Test the origin

2(0) + 3(0) ≤ 6 ✓

Step 4: Shade the side containing the origin

Answer: Shade the region below the line including the boundary
Graphing region idea
Draw the boundary line first, then test a point like the origin to know which side to shade.
Example 10: Find the region satisfying x + y ≤ 4, x ≥ 0, y ≥ 0

Step 1: Interpret the conditions

x ≥ 0 means the first quadrant or right side of the y-axis

y ≥ 0 means above the x-axis

x + y ≤ 4 means on or below the line x + y = 4

Step 2: Identify the feasible region

The solution is the triangular region in the first quadrant below the line.

Vertices: (0,0), (4,0), (0,4)

Answer: The feasible region is the triangle bounded by the axes and the line x + y = 4

5️⃣ Common Mistakes to Avoid

Mistake Correction
Forgetting to flip the sign when multiplying or dividing by a negative number Always reverse the inequality sign in that case
Including endpoints for strict inequalities like < or > Use open circles and parentheses
Choosing the wrong interval for a quadratic inequality Test points or sketch the parabola
Dividing by a variable without knowing whether it is positive or negative Bring all terms to one side instead

6️⃣ Exam Tips

How to score high in inequality questions

  • Always test your final interval with a sample value
  • Use number lines or quick sketches for quadratic inequalities
  • Remember the sign-flip rule very carefully
  • For graph questions, draw the boundary line before shading
  • For fractions, clear denominators using the LCM
🧠 Fast reminder

For quadratic inequalities, think “outside for positive, inside for negative” when the quadratic factorizes and opens upward.

7️⃣ Practice Questions

Q1. Solve: 3(x - 2) ≤ 2(x + 1) - 5

Step 1: Expand both sides

3x - 6 ≤ 2x + 2 - 5

3x - 6 ≤ 2x - 3

Step 2: Collect terms

x ≤ 3

Answer: x ≤ 3
Q2. Solve: x² - 7x + 10 < 0

Step 1: Factorize

(x - 5)(x - 2) < 0

Step 2: The expression is negative between the roots

2 < x < 5

Answer: (2, 5)
Q3. Find the range of values of x for which (x + 1)/2 - (x - 1)/3 > 1

Step 1: Find the LCM of 2 and 3

LCM = 6

Step 2: Multiply through by 6

3(x + 1) - 2(x - 1) > 6

Step 3: Expand and simplify

3x + 3 - 2x + 2 > 6

x + 5 > 6

x > 1

Answer: x > 1

8️⃣ Quick Checklist for Inequality Problems

🧠 Checklist

Identify whether the inequality is linear or quadratic. Remember to flip the sign when multiplying or dividing by a negative number. For quadratics, factorize and find the critical values. Use test points or a parabola sketch. For positive quadratic regions, take the outer intervals; for negative regions, take the inner interval. Check whether endpoints are included or excluded. For fractions, use the LCM to clear denominators.
