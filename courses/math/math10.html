🧮 Matrices and Determinants

JAMB Mathematics • matrix types • addition • subtraction • multiplication • determinants • inverse of 2×2 matrices

📋 Introduction to Chapter 10

This chapter teaches how to work with matrices and determinants. You will learn matrix types, matrix operations, how to calculate determinants, and how to find inverses of 2×2 matrices.

Matrix questions in JAMB are usually direct if you first check the matrix dimensions and then apply the correct formula carefully.

  • Matrices are arranged in rows and columns
  • Addition and subtraction require the same order
  • Matrix multiplication depends on matching inner dimensions
  • Only square matrices have determinants
  • A matrix has an inverse only if its determinant is not zero
🧠 Big idea

Most matrix mistakes happen when students ignore dimensions. Always check the order first before doing any operation.

❓ What is a Matrix?

A matrix is a rectangular array of numbers arranged in rows and columns and enclosed in brackets.

A = ( 1 2 3 ) ( 4 5 6 )

This is a 2 × 3 matrix because it has 2 rows and 3 columns.

2️⃣ Algebra of Matrices

A. Addition and Subtraction

Rule: Matrices must have the same order before you can add or subtract them.

Example 1: Matrix Addition
( 2 5 ) + ( 4 2 ) = ( 6 7 ) ( 3 1 ) ( 1 6 ) ( 4 7 )
Answer: ( 6 7 ; 4 7 )
Example 2: Matrix Subtraction
( 8 3 ) - ( 2 1 ) = ( 6 2 ) ( 5 9 ) ( 4 3 ) ( 1 6 )
Answer: ( 6 2 ; 1 6 )

B. Scalar Multiplication

Multiply every entry of the matrix by the scalar.

Example 3: Scalar Multiplication
3 × ( 2 4 ) = ( 6 12 ) ( 1 5 ) ( 3 15 )
Answer: ( 6 12 ; 3 15 )

C. Matrix Multiplication

Key rules

  • The number of columns in the first matrix must equal the number of rows in the second
  • The result has order: (rows of first matrix) × (columns of second matrix)
  • Multiply row by column, then add the products
Example 4: Multiply a 2×2 matrix by a 2×2 matrix
( 1 2 ) × ( 5 6 ) ( 3 4 ) ( 7 8 )

Row 1, Column 1: (1×5) + (2×7) = 19

Row 1, Column 2: (1×6) + (2×8) = 22

Row 2, Column 1: (3×5) + (4×7) = 43

Row 2, Column 2: (3×6) + (4×8) = 50

Result = ( 19 22 ) ( 43 50 )
Answer: ( 19 22 ; 43 50 )
Example 5: Multiply a 2×3 matrix by a 3×2 matrix
( 1 2 3 ) × ( 7 8 ) ( 4 5 6 ) ( 9 10 ) ( 11 12 )

Row 1, Column 1: (1×7)+(2×9)+(3×11)=58

Row 1, Column 2: (1×8)+(2×10)+(3×12)=64

Row 2, Column 1: (4×7)+(5×9)+(6×11)=139

Row 2, Column 2: (4×8)+(5×10)+(6×12)=154

Result = ( 58 64 ) ( 139 154 )
Answer: ( 58 64 ; 139 154 )

3️⃣ Determinants

What is a Determinant?

A determinant is a special number associated with a square matrix. It is written as |A| or det(A).

A. Determinant of a 2×2 Matrix

2×2 determinant formula
For A = ( a b ; c d )
|A| = ad - bc
Example 6: Find the determinant of ( 3 5 ; 2 4 )

|A| = (3×4) - (5×2)

= 12 - 10 = 2

Answer: 2

B. Determinant of a 3×3 Matrix

3×3 determinant formula
For A = ( a b c ; d e f ; g h i )
|A| = a(ei - fh) - b(di - fg) + c(dh - eg)
🧠 Memory tip

Expand along the first row and remember the sign pattern: +, −, +.

Example 7: Find the determinant of ( 1 2 3 ; 4 5 6 ; 7 8 9 )

|A| = 1(45 - 48) - 2(36 - 42) + 3(32 - 35)

= 1(-3) - 2(-6) + 3(-3)

= -3 + 12 - 9 = 0

Answer: 0

4️⃣ Inverse of a Matrix

Important: Only square matrices with non-zero determinants have inverses.

Inverse of a 2×2 Matrix

2×2 inverse formula
For A = ( a b ; c d )
A⁻¹ = (1/|A|) ( d -b ; -c a )
  • Find the determinant first
  • Swap the diagonal entries
  • Change the signs of the off-diagonal entries
  • Multiply everything by 1/|A|
Example 8: Find the inverse of A = ( 4 7 ; 2 6 )

Step 1: Find the determinant

|A| = (4×6) - (7×2) = 24 - 14 = 10

Step 2: Apply the formula

A⁻¹ = (1/10)( 6 -7 ; -2 4 )

Step 3: Write the final answer

A⁻¹ = ( 0.6 -0.7 ) ( -0.2 0.4 )

Or in fractions:

A⁻¹ = ( 3/5 -7/10 ) ( -1/5 2/5 )
Answer: A⁻¹ = (1/10)( 6 -7 ; -2 4 )

5️⃣ Using Inverse to Solve Simultaneous Equations

Example 9: Solve 2x + 3y = 8 and 5x + 4y = 13 using matrices

Step 1: Write in matrix form AX = B

( 2 3 ) ( x ) = ( 8 ) ( 5 4 ) ( y ) ( 13 )

Step 2: Find the determinant of A

|A| = (2×4) - (3×5) = 8 - 15 = -7

Step 3: Find A⁻¹

A⁻¹ = (1/-7)( 4 -3 ; -5 2 )

Step 4: Compute X = A⁻¹B

x = (-4/7)×8 + (3/7)×13 = (-32 + 39)/7 = 1

y = (5/7)×8 + (-2/7)×13 = (40 - 26)/7 = 2

Answer: x = 1, y = 2

7️⃣ JAMB Past Question Style

If A = ( 2 1 ; 3 4 ) and B = ( 1 2 ; 0 3 ), find AB and |AB|

Step 1: Multiply A and B

AB = ( (2×1 + 1×0) (2×2 + 1×3) ; (3×1 + 4×0) (3×2 + 4×3) )

AB = ( 2 7 ) ( 3 18 )

Step 2: Find the determinant

|AB| = (2×18) - (7×3) = 36 - 21 = 15

Answer: AB = ( 2 7 ; 3 18 ), and |AB| = 15

8️⃣ Key Points to Remember

Important facts

  • Matrix multiplication is usually not commutative, so AB ≠ BA in general
  • The determinant of the identity matrix is 1
  • If the determinant is 0, the matrix has no inverse
  • For 2×2 inverse, swap the diagonal and negate the off-diagonal
  • For 3×3 determinants, use alternating signs: +, −, +
  • Always check matrix order before operating

9️⃣ Quick Checklist for Matrix Problems

🧠 Checklist

Check the matrix dimensions first. For multiplication, make sure the number of columns of the first matrix equals the number of rows of the second. For determinants, the matrix must be square. For inverses, verify that |A| ≠ 0. For 2×2 inverse, swap the diagonal and change the signs of the off-diagonal entries. For 3×3 determinants, use the sign pattern +, −, +. For solving equations, use X = A⁻¹B.
