🎲 Probability

📋 Chapter Overview

This chapter covers the likelihood of events, types of events, and the major methods used to calculate probability.

Probability ranges from 0 for an impossible event to 1 for a certain event.

  • Probability compares favourable outcomes with total possible outcomes
  • Know the difference between mutually exclusive and independent events
  • “At least one” questions are usually faster with complements
  • Watch carefully for “with replacement” and “without replacement”
🧠 Big idea

Probability tells how likely an event is to happen, based on the total number of possible outcomes.

1️⃣ Basic Concepts

Definition and Key Terms

Basic probability formula
P(E) = Number of favourable outcomes / Total number of possible outcomes
Term Definition Example
Experiment An action with uncertain outcome Tossing a coin
Outcome A single result of an experiment Getting Head
Sample Space (S) Set of all possible outcomes S = {H, T}
Event (E) A subset of the sample space E = {H}

2️⃣ Types of Events

A. Mutually Exclusive Events

These are events that cannot happen at the same time.

Formula
P(A ∪ B) = P(A) + P(B)
Example 1: Mutually Exclusive

When a die is rolled, find the probability of getting 2 or 5.

P(2 or 5) = 1/6 + 1/6 = 2/6 = 1/3

Answer: 1/3

B. Independent Events

These are events where one event does not affect the other.

Formula
P(A ∩ B) = P(A) × P(B)
Example 2: Independent Events

A coin is tossed twice. Find the probability of getting two heads.

P(HH) = 1/2 × 1/2 = 1/4

Answer: 1/4

C. Complementary Events

The complement of an event means the event does not occur.

Formula
P(A') = 1 - P(A)
Example 3: Complementary Event

Find the probability of not getting 6 when a die is rolled.

P(not 6) = 1 - 1/6 = 5/6

Answer: 5/6

Very important shortcut: “At least one” is often easiest as 1 - P(none).

3️⃣ Calculation Methods

A. Simple Probability

Example 4: Ball Selection

A bag contains 5 red, 3 blue, and 2 green balls. There are 10 balls in total.

(a) Red ball

P(R) = 5/10 = 1/2

(b) Blue ball

P(B) = 3/10

(c) Not green

P(G') = 1 - 2/10 = 8/10 = 4/5

B. Addition Rule

For events that are not mutually exclusive, subtract the overlap.

Formula
P(A ∪ B) = P(A) + P(B) - P(A ∩ B)
Example 5: Addition Rule

In a class of 40 students, 20 study Maths, 15 study Physics, and 8 study both. Find P(Maths or Physics).

P(M ∪ P) = 20/40 + 15/40 - 8/40

= 27/40

Answer: 27/40

C. Conditional Probability

This is the probability of event A given that B has already occurred.

Formula
P(A|B) = P(A ∩ B) / P(B)
Example 6: Conditional Probability

Using the previous class data, find P(Physics | Maths).

P(P|M) = (8/40) / (20/40)

= 8/20 = 2/5

Answer: 2/5

4️⃣ Permutations and Combinations in Probability

Factorial Notation

Reminder
n! = n × (n−1) × (n−2) × ... × 1
5! = 120

Permutations

Use permutations when order matters.

Formula
nPr = n! / (n−r)!
Example 7: Permutation

Arrange 3 people from 5 in a row.

5P3 = 5! / (5−3)! = 5!/2! = 120/2 = 60

Answer: 60 ways

Combinations

Use combinations when order does not matter.

Formula
nCr = n! / [r!(n−r)!]
Example 8: Combination

Choose 3 students from 10 for a committee.

10C3 = 10! / (3! × 7!)

= (10 × 9 × 8) / (3 × 2 × 1) = 720/6 = 120

Answer: 120 ways

5️⃣ Advanced Examples

Example 9: With Replacement

A bag has 4 white and 6 black balls. Two balls are drawn with replacement. Find the probability that both are white.

P(WW) = 4/10 × 4/10 = 16/100 = 4/25

Answer: 4/25
Example 10: Without Replacement

Using the same bag, find the probability that both are white without replacement.

P(WW) = 4/10 × 3/9 = 12/90 = 2/15

Answer: 2/15
Example 11: At Least One

Find the probability of getting at least one 6 when rolling two dice.

Use complement:

1 - P(no sixes)

= 1 - (5/6 × 5/6)

= 1 - 25/36 = 11/36

Answer: 11/36

📋 Complete Formula Summary

Concept Formula When to Use
Basic Probability P(E) = n(E)/n(S) Simple events
Complementary P(E') = 1 - P(E) “Not” happening
Mutually Exclusive P(A ∪ B) = P(A) + P(B) A or B, but not both
Addition Rule P(A ∪ B) = P(A) + P(B) - P(A ∩ B) A or B, overlap possible
Independent Events P(A ∩ B) = P(A) × P(B) A and B independent
Conditional Probability P(A|B) = P(A ∩ B)/P(B) A given B occurred
Permutations nPr = n!/(n−r)! Arrangements
Combinations nCr = n!/[r!(n−r)!] Selections
Expected Value E(X) = ∑x · P(x) Average outcome

✏️ Practice Question

Three Coins Tossed Together

Three coins are tossed simultaneously. Find the probability of:

(a) Exactly two heads

(b) At least one tail

(c) No heads

Sample space:

{HHH, HHT, HTH, THH, HTT, THT, TTH, TTT}

Total outcomes = 8

Answers: (a) 3/8, (b) 7/8, (c) 1/8

🎯 JAMB Exam Tips

Best strategy

  • Read carefully to notice “with replacement” or “without replacement”
  • Draw tree diagrams for more complex problems
  • Use 1 - P(none) for “at least one” questions
  • Check whether the events are independent before multiplying
  • Use combinations for probability questions involving selection
  • Identify the sample space clearly before solving
⚠️ Common mistakes

Do not confuse replacement with non-replacement. Do not forget to subtract the intersection in the addition rule. Do not assume events are independent when they are not. Also simplify final fractions fully.
