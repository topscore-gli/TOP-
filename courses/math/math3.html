📘 Indices, Logarithms & Surds

JAMB Mathematics • powers • logarithmic equations • surd operations • rationalization • worked examples

📋 Introduction to Chapter 3

This chapter covers indices, logarithms, and surds. These topics appear frequently in JAMB because they test both manipulation skills and understanding of rules.

To do well, you need to know the laws, apply them carefully, and avoid common mistakes like adding logarithms incorrectly or forgetting to rationalize a denominator.

  • Indices questions often become easy when all terms are written in the same base
  • Logarithms are only defined for positive values
  • Surds should be simplified fully in final answers
  • Conjugates are important in rationalization
🧠 Fast memory link

Indices deal with powers, logarithms reverse powers, and surds come from fractional powers. So these three topics are strongly connected.

1️⃣ Indices (Powers / Exponents)

Basic Laws of Indices

Law Formula Example
Multiplication aᵐ × aⁿ = aᵐ⁺ⁿ 2³ × 2⁴ = 2⁷ = 128
Division aᵐ ÷ aⁿ = aᵐ⁻ⁿ 5⁶ ÷ 5² = 5⁴ = 625
Power of Power (aᵐ)ⁿ = aᵐⁿ (3²)³ = 3⁶ = 729
Zero Index a⁰ = 1, a ≠ 0 7⁰ = 1
Negative Index a⁻ⁿ = 1/aⁿ 2⁻³ = 1/8
Fractional Index a^(1/n) = ⁿ√a 8^(1/3) = 2
General Fractional Index a^(m/n) = (ⁿ√a)^m 16^(3/4) = 2³ = 8

Example 1: Simplify (3⁵ × 3⁻²) / 3³

Step 1: Add powers in the numerator

3⁵ × 3⁻² = 3^(5 + -2) = 3³

Step 2: Divide using subtraction of powers

3³ / 3³ = 3^(3 - 3) = 3⁰ = 1

Answer: 1
Example 2: Solve 2^(2x+1) = 8^(x-1)

Step 1: Express 8 in base 2

8 = 2³, so 8^(x-1) = (2³)^(x-1) = 2^(3x-3)

Step 2: Equate powers because the bases are the same

2x + 1 = 3x - 3

Step 3: Solve

x = 4

Answer: x = 4
Example 3: Evaluate 27^(2/3) + 16^(3/4)

Step 1: Evaluate the first term

27^(2/3) = (³√27)² = 3² = 9

Step 2: Evaluate the second term

16^(3/4) = (⁴√16)³ = 2³ = 8

Step 3: Add

9 + 8 = 17

Answer: 17
🧠 Indices shortcut

When solving equations, always try to rewrite all numbers using one common base. That usually makes the question much shorter.

2️⃣ Logarithms

Definition

If aˣ = N, then logₐN = x.

  • a is the base and must be positive and not equal to 1
  • N must be positive
  • x is the logarithm

Laws of Logarithms

Law Formula Example
Product Rule logₐ(MN) = logₐM + logₐN log₂(8×4) = 3 + 2 = 5
Quotient Rule logₐ(M/N) = logₐM - logₐN log₃(27/9) = 3 - 2 = 1
Power Rule logₐ(Mⁿ) = n logₐM log₅(25²) = 2 × 2 = 4
Change of Base logₐb = (logcb)/(logca) log₂8 = log8 / log2
Log of Base logₐa = 1 log₅5 = 1
Log of 1 logₐ1 = 0 log₇1 = 0

Common and Natural Logarithms

Common logarithms are base 10 and are usually written simply as log.

Natural logarithms are base e and written as ln, where e ≈ 2.718.

Example 4: Simplify log₃54 + log₃2 - log₃4

Step 1: Use the product rule

log₃54 + log₃2 = log₃(54 × 2) = log₃108

Step 2: Use the quotient rule

log₃108 - log₃4 = log₃(108/4) = log₃27

Step 3: Evaluate

27 = 3³, so log₃27 = 3

Answer: 3
Example 5: Solve log₂(x + 3) + log₂(x - 1) = 3

Step 1: Combine the logarithms

log₂[(x + 3)(x - 1)] = 3

Step 2: Convert to exponential form

(x + 3)(x - 1) = 2³ = 8

Step 3: Expand

x² + 2x - 3 = 8

x² + 2x - 11 = 0

Step 4: Use quadratic formula

x = (-2 ± √48)/2 = -1 ± 2√3

Step 5: Check domain

Since x - 1 > 0, we need x > 1. So only the positive value is valid.

Answer: x = -1 + 2√3 ≈ 2.46
Example 6: Evaluate log₄8

Step 1: Use change of base

log₄8 = (log₂8) / (log₂4)

Step 2: Evaluate

= 3/2 = 1.5

Answer: 1.5

Very important log warning

  • logₐ(M + N) ≠ logₐM + logₐN
  • The log laws work for products, quotients, and powers — not sums

3️⃣ Surds (Irrational Roots)

Definition

Surds are irrational numbers expressed as roots that cannot be simplified into exact rational numbers.

Examples include √2, √3, ³√5.

Types of Surds

Type Description Example
Quadratic Surd Square root of a non-perfect square √5, √18
Cubic Surd Cube root of a non-perfect cube ³√2, ³√7
Binomial Surd Sum or difference of two surds 3 + √2, √5 - √3
Conjugate Surd Changes the sign between two terms If a + √b, conjugate is a - √b

Operations with Surds

Simplification rules
√(a × b) = √a × √b
√(a² × b) = a√b

Example idea: √50 = √(25 × 2) = 5√2

Rationalization

Rationalization means removing surds from the denominator.

Form Multiply By Result
1/√a √a / √a √a / a
1/(a + √b) (a - √b)/(a - √b) (a - √b)/(a² - b)
1/(√a + √b) (√a - √b)/(√a - √b) (√a - √b)/(a - b)

Example 7: Simplify √72 + √50 - √32

Step 1: Simplify each surd

√72 = √(36 × 2) = 6√2

√50 = √(25 × 2) = 5√2

√32 = √(16 × 2) = 4√2

Step 2: Add like surds

6√2 + 5√2 - 4√2 = 7√2

Answer: 7√2
Example 8: Rationalize 3 / (2 + √5)

Step 1: Multiply by the conjugate (2 - √5)

[3/(2 + √5)] × [(2 - √5)/(2 - √5)]

Step 2: Simplify denominator

(2 + √5)(2 - √5) = 4 - 5 = -1

Step 3: Simplify numerator

3(2 - √5) = 6 - 3√5

Step 4: Divide by -1

(6 - 3√5)/(-1) = 3√5 - 6

Answer: 3√5 - 6
Example 9: If a = 3 + 2√2 and b = 3 - 2√2, find a² + b²

Step 1: Find a + b and ab

a + b = 6

ab = (3 + 2√2)(3 - 2√2) = 9 - 8 = 1

Step 2: Use identity

a² + b² = (a + b)² - 2ab

= 6² - 2(1) = 36 - 2 = 34

Answer: 34
Conjugate idea
Conjugates have the same terms but opposite signs. Their product removes the surd in the middle term.

5️⃣ JAMB Exam Tips

How to score high in this chapter

  • For indices, express numbers in the same base whenever possible
  • For logarithms, always check that the expression inside the log is positive
  • For surds, simplify fully before giving the final answer
  • When rationalizing a binomial denominator, use the conjugate
  • Never apply log laws to addition or subtraction inside brackets

Common pitfall: logₐ(M + N) is not equal to logₐM + logₐN. This is false and causes many avoidable mistakes.

6️⃣ Practice Questions

Q1: Simplify [25^(1/2) × 8^(2/3)] / 16^(3/4)

Step 1: Evaluate each term

25^(1/2) = 5

8^(2/3) = (³√8)² = 2² = 4

16^(3/4) = (⁴√16)³ = 2³ = 8

Step 2: Simplify

(5 × 4) / 8 = 20/8 = 5/2 = 2.5

Answer: 5/2 or 2.5
Q2: Solve log₃x + log₃(x - 2) = 1

Step 1: Combine logs

log₃[x(x - 2)] = 1

Step 2: Convert to exponential form

x(x - 2) = 3

Step 3: Expand and solve

x² - 2x - 3 = 0

(x - 3)(x + 1) = 0

Step 4: Check valid solution

x = 3 or x = -1, but x - 2 must be positive.

Answer: x = 3
Q3: Rationalize (√5 + √3) / (√5 - √3)

Step 1: Multiply by the conjugate of the denominator

[(√5 + √3)/(√5 - √3)] × [(√5 + √3)/(√5 + √3)]

Step 2: Simplify denominator

(√5 - √3)(√5 + √3) = 5 - 3 = 2

Step 3: Simplify numerator

(√5 + √3)² = 5 + 3 + 2√15 = 8 + 2√15

Step 4: Divide by 2

(8 + 2√15)/2 = 4 + √15

Answer: 4 + √15
