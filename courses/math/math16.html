📈 Applications of Differentiation

JAMB Mathematics • rate of change • related rates • maxima and minima • turning points • optimization

📋 Introduction

This chapter explains the two most important applications of differentiation for JAMB:

  • Rate of Change
  • Maxima and Minima

The ideas are simple once you know what derivative means in each situation.

  • dy/dx gives the rate at which y changes with respect to x
  • If s is displacement, then ds/dt is velocity
  • If v is velocity, then dv/dt is acceleration
  • For maxima and minima, first solve dy/dx = 0
  • Then use the second derivative to decide the nature of the point
🧠 Quick recap

Rate of Change: derivative = instantaneous rate of change. Maxima and Minima: stationary points come from dy/dx = 0, then d²y/dx² tells whether it is a maximum or minimum.

📊 Core Summary Table

1️⃣ Rate of Change

Easy Idea

The derivative tells us how fast something changes. If the function depends on time t, then differentiate with respect to t.

Important formulas
v = ds/dt
a = dv/dt = d²s/dt²
If A = πr², then dA/dt = 2πr dr/dt
If V = (4/3)πr³, then dV/dt = 4πr² dr/dt
Rate of change as slope of tangent
At a particular point, the derivative is the slope of the tangent line.

10 JAMB-Style Examples

Example 1: Sphere Volume with respect to Radius

The volume of a sphere is V = (4/3)πr³. Find dV/dr when r = 3 cm.

dV/dr = 4πr²

At r = 3:

dV/dr = 4π(9) = 36π

Answer: 36π
Example 2: Velocity from Displacement

A particle moves so that s = 4t³ – 6t² + 5t + 2. Find its velocity at t = 1.

v = ds/dt = 12t² – 12t + 5

At t = 1:

v = 12 - 12 + 5 = 5 m/s

Answer: 5 m/s
Example 3: Area of Circle Increasing

The area of a circle increases at 4 cm²/s. Find dr/dt when r = 5 cm. Take π ≈ 3.14.

A = πr²

dA/dt = 2πr dr/dt

4 = 2 × 3.14 × 5 × dr/dt

4 = 31.4 dr/dt

dr/dt ≈ 4 / 31.4 ≈ 0.128 cm/s

Answer: approximately 0.128 cm/s
Example 4: Area of a Square

The side of a square increases at 2 cm/s. Find the rate of increase of the area when the side is 8 cm.

A = x²

dA/dt = 2x dx/dt

At x = 8 and dx/dt = 2:

dA/dt = 2(8)(2) = 32 cm²/s

Answer: 32 cm²/s
Example 5: Acceleration

Given s = t⁴ – 8t³ + 12t², find the acceleration when t = 2.

v = ds/dt = 4t³ – 24t² + 24t

a = dv/dt = 12t² – 48t + 24

At t = 2:

a = 12(4) - 48(2) + 24 = 48 - 96 + 24 = -24

Answer: -24 units/s²
Example 6: Conical Tank

Water flows into a conical tank at 20 cm³/s. The cone has radius r = h/3. Find dh/dt when h = 9 cm.

V = (1/3)πr²h

Since r = h/3,

V = (1/3)π(h/3)²h = (π/27)h³

dV/dt = (π/9)h² dh/dt = 20

At h = 9:

(π/9)(81) dh/dt = 20

9π dh/dt = 20

dh/dt = 20/(9π)

Answer: 20/(9π) cm/s
Example 7: Balloon Radius Increasing

The radius of a balloon increases at 0.4 cm/s. Find dV/dt when r = 10 cm.

V = (4/3)πr³

dV/dt = 4πr² dr/dt

dV/dt = 4π(10²)(0.4) = 160π

Answer: 160π cm³/s
Example 8: Ladder Problem

A 17 m ladder slides down a wall. The base moves away at 1.5 m/s. Find how fast the top descends when the base is 8 m from the wall.

x² + y² = 17² = 289

Differentiate:

2x dx/dt + 2y dy/dt = 0

At x = 8,

y = √(289 - 64) = √225 = 15

2(8)(1.5) + 2(15) dy/dt = 0

24 + 30 dy/dt = 0

dy/dt = -24/30 = -0.8 m/s

Answer: -0.8 m/s
Example 9: Cube Volume

The volume of a cube increases at 54 cm³/s. Find the rate of increase of its side when the side is 6 cm.

V = x³

dV/dt = 3x² dx/dt

54 = 3(6²) dx/dt = 108 dx/dt

dx/dt = 54/108 = 0.5 cm/s

Answer: 0.5 cm/s
Example 10: Differentiate and Evaluate

If y = 5x⁴ – 12x³ + 8x², find dy/dx at x = 1.

dy/dx = 20x³ – 36x² + 16x

At x = 1:

20 - 36 + 16 = 0

Answer: 0

2️⃣ Maxima and Minima

Easy Idea

To find the maximum or minimum value of a function:

  1. Differentiate the function
  2. Set dy/dx = 0 to find critical points
  3. Use the second derivative test
Second derivative test
dy/dx = 0 gives critical points
d²y/dx² > 0 means minimum
d²y/dx² < 0 means maximum
Maxima and minima on a curve
A turning point may be a maximum or a minimum depending on the sign of the second derivative.

10 JAMB-Style Examples

Example 1: Maximum Value of x(6-x)

What value of x gives a maximum for y = x(6-x)?

y = 6x - x²

dy/dx = 6 - 2x

Set equal to zero:

6 - 2x = 0 → x = 3

d²y/dx² = -2 < 0

Answer: 3
Example 2: Minimum Value

Find the minimum value of y = x² + 4x + 7.

dy/dx = 2x + 4

2x + 4 = 0 → x = -2

d²y/dx² = 2 > 0, so this is a minimum.

Substitute back:

y = (-2)² + 4(-2) + 7 = 4 - 8 + 7 = 3

Answer: 3
Example 3: Rectangle of Maximum Area

A rectangle has perimeter 80 m. Find the dimensions for maximum area.

2(l+w) = 80 so l+w = 40

w = 40 - l

A = lw = l(40-l) = 40l - l²

dA/dl = 40 - 2l

40 - 2l = 0 → l = 20

w = 20

d²A/dl² = -2 < 0

Answer: 20 m by 20 m
Example 4: Maximum Profit

P = 120x - 2x² - 400. Find the number of items for maximum profit.

dP/dx = 120 - 4x

120 - 4x = 0 → x = 30

d²P/dx² = -4 < 0

Answer: 30 items
Example 5: Maximum Value of a Quadratic

Find the maximum value of y = 8x - x² - 10.

dy/dx = 8 - 2x

8 - 2x = 0 → x = 4

d²y/dx² = -2 < 0

Substitute x = 4:

y = 8(4) - 16 - 10 = 6

Answer: 6
Example 6: Coordinates of Maximum and Minimum Points

For y = x³ - 6x² + 9x + 2, find the coordinates of the maximum and minimum points.

dy/dx = 3x² - 12x + 9

3x² - 12x + 9 = 0

x² - 4x + 3 = 0

(x-1)(x-3) = 0

So x = 1 or x = 3

d²y/dx² = 6x - 12

At x = 1: 6 - 12 = -6 < 0, so maximum

y = 1 - 6 + 9 + 2 = 6

At x = 3: 18 - 12 = 6 > 0, so minimum

y = 27 - 54 + 27 + 2 = 2

Answer: Maximum at (1, 6), Minimum at (3, 2)
Example 7: Output for Maximum Profit

A company’s cost function is C(x) = x² + 40x + 500, and revenue is R(x) = 80x. Find the output for maximum profit.

P = R - C = 80x - x² - 40x - 500

P = -x² + 40x - 500

dP/dx = -2x + 40

-2x + 40 = 0 → x = 20

d²P/dx² = -2 < 0

Answer: 20
Example 8: Turning Points and Their Nature

Find the turning points of y = 2x³ - 9x² + 12x - 5 and state their nature.

dy/dx = 6x² - 18x + 12

6x² - 18x + 12 = 0

x² - 3x + 2 = 0

(x-1)(x-2) = 0

So x = 1 or x = 2

d²y/dx² = 12x - 18

At x = 1: 12 - 18 = -6 < 0, so maximum

At x = 2: 24 - 18 = 6 > 0, so minimum

Answer: Maximum at x = 1, Minimum at x = 2
Example 9: Maximum Value of Another Quadratic

The function y = -4x² + 24x - 35 has a maximum value. Find it.

dy/dx = -8x + 24

-8x + 24 = 0 → x = 3

d²y/dx² = -8 < 0

Substitute back:

y = -4(9) + 24(3) - 35 = -36 + 72 - 35 = 1

Answer: 1
Example 10: Open Box of Minimum Surface Area

An open-top box with square base has volume 32 cm³. Find the base side for minimum surface area.

Let base side = x, height = h

x²h = 32 so h = 32/x²

Surface area:

A = x² + 4xh

A = x² + 4x(32/x²) = x² + 128/x

dA/dx = 2x - 128/x²

Set equal to zero:

2x - 128/x² = 0

2x³ = 128

x³ = 64

x = 4 cm

d²A/dx² = 2 + 256/x³ > 0

Answer: 4 cm

💡 JAMB Exam Tips

Best way to approach these questions

  • Read carefully to know whether the question is asking for a rate or an optimum value
  • For motion questions, remember displacement → velocity → acceleration
  • For related rates, write the original formula first before differentiating
  • For maxima and minima, do not stop at dy/dx = 0 — test the nature
  • In word problems, first express the quantity in one variable
  • Always substitute the final critical value back into the original expression if a maximum or minimum value is required
  • Rate of change: derivative tells how fast something changes
  • Velocity: ds/dt
  • Acceleration: d²s/dt²
  • Maximum: d²y/dx² < 0
  • Minimum: d²y/dx² > 0

✅ Pre-Exam Checklist

🧠 Checklist

Make sure you can differentiate polynomials confidently. Know the meaning of ds/dt, dv/dt, and related rates. Revise formulas for area and volume because they often appear in rate-of-change problems. Practise setting dy/dx = 0 for maxima and minima. Use the second derivative test correctly. Also practise optimization questions involving area, volume, cost, and profit.
