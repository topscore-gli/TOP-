📚 Introduction to Integration

JAMB Mathematics • reverse of differentiation • basic integrals • substitution • parts • definite integrals • area and volume

📋 What is Integration?

Integration is the reverse process of differentiation. If differentiation gives the rate of change, integration helps us find the original function from that rate of change.

  • Indefinite Integration: general function and always includes + C
  • Definite Integration: gives a numerical value between limits
  • Constant of Integration: needed because the derivative of any constant is zero

JAMB focus

  • Expect about 3–5 questions on integration
  • Questions may combine algebraic, trigonometric, and application problems
🧠 Big idea

Differentiation gives the slope or rate. Integration helps us rebuild the original expression from that derivative.

1️⃣ Basic Integration Rules

Power Rule

Most important rule
∫ xⁿ dx = xⁿ⁺¹/(n+1) + C, where n ≠ -1
∫ k·f(x) dx = k ∫ f(x) dx
∫ [f(x) ± g(x)] dx = ∫ f(x) dx ± ∫ g(x) dx
Example 1: Power Rule

Find ∫ 3x⁴ dx.

∫ 3x⁴ dx = 3 · x⁵/5 + C

= 3x⁵/5 + C

Answer: 3x⁵/5 + C
Example 2: Polynomial

Find ∫ (2x³ - 5x² + 4x - 7) dx.

Integrate term by term:

= 2·x⁴/4 - 5·x³/3 + 4·x²/2 - 7x + C

= x⁴/2 - 5x³/3 + 2x² - 7x + C

Answer: x⁴/2 - 5x³/3 + 2x² - 7x + C

2️⃣ Algebraic Functions

Rational and Exponential Functions

Special cases
∫ 1/x dx = ln|x| + C
∫ eˣ dx = eˣ + C
∫ aˣ dx = aˣ/ln a + C
Example 3: Rational Function

Find ∫ 3/x dx.

∫ 3/x dx = 3 ln|x| + C

Answer: 3ln|x| + C

Example 4: Trigonometric Integral

Find ∫ (4 sin x + 3 cos x) dx.

= 4(-cos x) + 3(sin x) + C

= -4 cos x + 3 sin x + C

Answer: -4 cos x + 3 sin x + C
Example 5: Using Identity

Find ∫ sin² x dx.

Use sin² x = (1 - cos 2x)/2

∫ sin² x dx = ∫ (1 - cos 2x)/2 dx

= 1/2∫1 dx - 1/2∫cos 2x dx

= x/2 - sin 2x/4 + C

Answer: x/2 - sin 2x/4 + C

4️⃣ Integration by Substitution

When to Use

Use substitution when a function appears together with its derivative, or when you can simplify a composite expression by replacing the inner part.

Method
1. Let u = inner function
2. Find du
3. Substitute into the integral
4. Integrate with respect to u
5. Substitute back to x
Example 6: Substitution

Find ∫ 2x(x² + 3)⁴ dx.

Let u = x² + 3

Then du = 2x dx

∫ 2x(x² + 3)⁴ dx = ∫ u⁴ du

= u⁵/5 + C

= (x² + 3)⁵/5 + C

Answer: (x² + 3)⁵/5 + C
Example 7: Trigonometric Substitution

Find ∫ cos 3x dx.

Let u = 3x

Then du = 3 dx, so dx = du/3

∫ cos 3x dx = ∫ cos u · du/3

= (1/3) sin u + C

= sin 3x/3 + C

Answer: sin 3x/3 + C

5️⃣ Integration by Parts

Formula and LIATE Rule

Formula
∫ u dv = uv - ∫ v du
  • LIATE helps you choose u
  • Logarithmic
  • Inverse trigonometric
  • Algebraic
  • Trigonometric
  • Exponential
Example 8: Integration by Parts

Find ∫ x eˣ dx.

Let u = x, dv = eˣ dx

Then du = dx, v = eˣ

∫ x eˣ dx = xeˣ - ∫ eˣ dx

= xeˣ - eˣ + C

= eˣ(x - 1) + C

Answer: eˣ(x - 1) + C
Example 9: Another Parts Example

Find ∫ x ln x dx.

Let u = ln x, dv = x dx

Then du = (1/x)dx, v = x²/2

∫ x ln x dx = (x²/2)ln x - ∫ (x²/2)(1/x) dx

= (x²/2)ln x - ∫ x/2 dx

= (x²/2)ln x - x²/4 + C

= (x²/4)(2ln x - 1) + C

Answer: (x²/4)(2ln x - 1) + C

6️⃣ Definite Integration

Format and Properties

Main formula
∫ₐᵇ f(x) dx = [F(x)]ₐᵇ = F(b) - F(a)
  • ∫ₐᵇ f(x) dx = -∫ᵇₐ f(x) dx
  • ∫ₐₐ f(x) dx = 0
  • ∫ₐᵇ f(x) dx + ∫ᵇᶜ f(x) dx = ∫ₐᶜ f(x) dx
Example 10: Definite Integral

Evaluate ∫₁³ (2x + 1) dx.

∫ (2x + 1) dx = x² + x

[x² + x]₁³ = (3² + 3) - (1² + 1)

= 12 - 2 = 10

Answer: 10
Example 11: Definite Integral with Substitution

Evaluate ∫₀² x(x² + 1)³ dx.

Let u = x² + 1, so du = 2x dx and x dx = du/2

When x = 0, u = 1

When x = 2, u = 5

∫₀² x(x² + 1)³ dx = ∫₁⁵ u³ · du/2

= 1/2 [u⁴/4]₁⁵

= 1/8(5⁴ - 1⁴)

= 1/8(625 - 1) = 624/8 = 78

Answer: 78

7️⃣ Applications of Integration

A. Area Under a Curve

Area formula
Area = ∫ₐᵇ y dx when the curve is above the x-axis
Example 12: Area Under a Curve

Find the area bounded by y = x², x = 1, x = 3, and the x-axis.

Area = ∫₁³ x² dx

= [x³/3]₁³

= 27/3 - 1/3 = 26/3

Answer: 26/3 square units
Area under a curve
Sketching helps you see the region and choose correct limits.

B. Area Between Two Curves

Area between curves
Area = ∫ₐᵇ (yupper - ylower) dx
Example 13: Area Between Two Curves

Find the area enclosed by y = x² and y = x.

Find the points of intersection:

x² = x → x = 0, 1

Area = ∫₀¹ (x - x²) dx

= [x²/2 - x³/3]₀¹

= 1/2 - 1/3 = 1/6

Answer: 1/6 square units

C. Volume of Revolution

About the x-axis
V = π∫ₐᵇ y² dx
Example 14: Volume of Revolution

Find the volume generated by rotating y = √x from x = 0 to x = 4 about the x-axis.

V = π∫₀⁴ (√x)² dx = π∫₀⁴ x dx

= π[x²/2]₀⁴

= π(16/2) = 8π

Answer: 8π cubic units

🎯 Practice Tips for JAMB

Best strategy

  • Master the basics first: power rule and standard integrals
  • Practise substitution because it is one of the most common JAMB techniques
  • Watch your signs carefully, especially in trigonometric integrals
  • Differentiate your answer to verify it
  • Time yourself and aim for 2–3 minutes per question
  • Sketch graphs first for area questions
