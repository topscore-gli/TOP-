⚙️ Binary Operations

JAMB Mathematics • defined operations • closure • commutative • associative • identity • inverse • distributive property

📋 Introduction to Chapter 9

This chapter explains binary operations and how to test their properties. In JAMB, you are usually given a defined operation and asked to evaluate expressions, find identity or inverse elements, or check whether the operation is commutative or associative.

These questions are not difficult once you write the operation clearly and substitute carefully.

  • A binary operation combines two elements of a set to give another element of the same set
  • The rule may use symbols like *, ⊕, ⊗, ∘, or any special symbol
  • Always write the rule first before solving
  • Identity and inverse questions are very common in JAMB
🧠 Big idea

Binary operations are really just custom rules. The main job is to follow the rule accurately and then test the required property step by step.

❓ What is a Binary Operation?

A binary operation is a rule that combines two elements of a set to produce one element from the same set.

Definition
If S is a set and * is an operation, then for any a, b ∈ S:
a * b = c, where c ∈ S

Common symbols: *, ∘, ⊕, ⊗, Δ, ∇, or any rule specially defined in the question.

1️⃣ Properties of Binary Operations

A. Closure Property

A set S is closed under an operation * if whenever a, b ∈ S, then a * b ∈ S.

Closure Example

Let the set of even numbers be E = {2, 4, 6, 8, ...} under addition.

4 + 6 = 10, and 10 ∈ E

Therefore, the set of even numbers is closed under addition.

B. Commutative Property

Condition
a * b = b * a
Commutative Example

Operation: a * b = a + b + ab

2 * 3 = 2 + 3 + (2)(3) = 11

3 * 2 = 3 + 2 + (3)(2) = 11

Therefore, the operation is commutative.

C. Associative Property

Condition
(a * b) * c = a * (b * c)
Associative Example

Operation: a * b = a + b - 1

LHS: (2 * 3) * 4

= (2 + 3 - 1) * 4 = 4 * 4 = 4 + 4 - 1 = 7

RHS: 2 * (3 * 4)

= 2 * (3 + 4 - 1) = 2 * 6 = 2 + 6 - 1 = 7

Since LHS = RHS, the operation is associative.

D. Identity Element

Condition
a * e = e * a = a
Finding Identity Example

Operation: a * b = a + b - 5

Find e such that a * e = a

a + e - 5 = a

e = 5

Identity element = 5

E. Inverse Element

Condition
For an element a, its inverse satisfies a * a⁻¹ = e
Finding Inverse Example

Operation: a * b = a + b - 5, identity e = 5

Find the inverse of 3

3 * x = 5

3 + x - 5 = 5

x - 2 = 5

x = 7

Inverse of 3 is 7

F. Distributive Property

Condition
a ∘ (b * c) = (a ∘ b) * (a ∘ c)

2️⃣ Worked Examples (JAMB Style)

Example 1: If a * b = (a + b)/2, evaluate 4 * 6 and (2 * 8) * 4

(a) Evaluate 4 * 6

4 * 6 = (4 + 6)/2 = 10/2 = 5

(b) Evaluate (2 * 8) * 4

First, 2 * 8 = (2 + 8)/2 = 5

Then, 5 * 4 = (5 + 4)/2 = 9/2 = 4.5

Answers: 5 and 4.5
Example 2: For a * b = a + b + ab, find the identity element

Let the identity be e

a * e = a

a + e + ae = a

e + ae = 0

e(1 + a) = 0

For this to hold for all a, e = 0

Verification: a * 0 = a + 0 + a(0) = a

Answer: Identity = 0
Example 3: Using a * b = a + b + ab with identity 0, find the inverse of 2

Let the inverse of 2 be x

2 * x = 0

2 + x + 2x = 0

2 + 3x = 0

3x = -2

x = -2/3

Answer: Inverse of 2 is -2/3
Example 4: For a * b = 2a + b - 3, test if the operation is commutative and associative

(a) Commutative?

a * b = 2a + b - 3

b * a = 2b + a - 3

Since these are not always equal, the operation is not commutative.

(b) Associative?

(a * b) * c = (2a + b - 3) * c

= 2(2a + b - 3) + c - 3 = 4a + 2b + c - 9

a * (b * c) = a * (2b + c - 3)

= 2a + (2b + c - 3) - 3 = 2a + 2b + c - 6

Since these are not always equal, the operation is not associative.

Answer: Not commutative and not associative
Example 5: Modulo Operation on {0,1,2,3,4} defined by a ⊕ b = remainder when a + b is divided by 5

4️⃣ Quick Reference: Solving Steps

  • Evaluate a * b: Substitute values into the given rule
  • Check commutative: Compare a * b and b * a
  • Check associative: Compare (a * b) * c and a * (b * c)
  • Find identity: Solve a * e = a
  • Find inverse: Solve a * x = e
  • Construct table: Fill entries systematically row by row

5️⃣ JAMB Exam Tips

How to score high in binary operation questions

  • Always rewrite the operation definition clearly before working
  • Substitute carefully and simplify step by step
  • For identity and inverse, verify by substitution after solving
  • Modulo questions usually involve remainders smaller than the divisor
  • When filling tables, move systematically across rows and columns
🧠 Fast reminder

Most mistakes in binary operations come from using the wrong rule or skipping a substitution step. Slow, neat substitution usually wins here.

6️⃣ Practice Question

An operation * is defined on the set of real numbers by a * b = ab/4

(a) Evaluate 6 * 8

6 * 8 = (6 × 8)/4 = 48/4 = 12

(b) Find the identity element

Let the identity be e

a * e = a

ae/4 = a

e = 4

(c) Find the inverse of 2

We need 2 * x = 4 because the identity is 4

(2x)/4 = 4

2x = 16

x = 8

(d) Is the operation commutative?

a * b = ab/4 = ba/4 = b * a

Answers: (a) 12, (b) 4, (c) 8, (d) Yes, it is commutative

7️⃣ Quick Checklist for Binary Operations

🧠 Checklist

Write down the operation definition clearly. For evaluation, substitute carefully. For properties, compare both required expressions. For identity, solve a * e = a. For inverse, solve a * x = e. Always verify your answer by substitution. For modulo operations, remember the remainder is always less than the divisor.
