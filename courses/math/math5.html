1️⃣ Understanding Polynomials

What is a Polynomial?

A polynomial is an expression made up of variables and coefficients, involving only addition, subtraction, multiplication, and non-negative integer powers of the variable.

General form
P(x) = aₙxⁿ + aₙ₋₁xⁿ⁻¹ + ... + a₁x + a₀
aₙ, aₙ₋₁, ... are coefficients
x is the variable
n is the degree
a₀ is the constant term

Types by Degree

Degree Name General Form Example
0 Constant P(x) = a P(x) = 5
1 Linear P(x) = ax + b P(x) = 2x + 3
2 Quadratic P(x) = ax² + bx + c P(x) = x² - 4x + 4
3 Cubic P(x) = ax³ + bx² + cx + d P(x) = 2x³ - x² + 3x - 1

2️⃣ Change of Subject of Formula

Making a Variable the Subject

This means rearranging the formula until the required variable stands alone.

Example 1: Make x the subject in y = 3x + 7

Step 1: Subtract 7 from both sides

y - 7 = 3x

Step 2: Divide by 3

x = (y - 7)/3

Answer: x = (y - 7)/3
Example 2: Make x the subject in y = 2x² - 8

Step 1: Add 8 to both sides

y + 8 = 2x²

Step 2: Divide by 2

x² = (y + 8)/2

Step 3: Take square root

x = ±√[(y + 8)/2]

Answer: x = ±√[(y + 8)/2]
Example 3: Make x the subject in T = 2π√(x/g)

Step 1: Divide by 2π

T/(2π) = √(x/g)

Step 2: Square both sides

(T/2π)² = x/g

Step 3: Multiply by g

x = g(T/2π)² = gT² / 4π²

Answer: x = gT² / 4π²

3️⃣ Multiplication and Division

Multiplication of Polynomials

Example 4: Multiply (2x + 3)(x² - 2x + 4)

Step 1: Expand each part

= 2x(x² - 2x + 4) + 3(x² - 2x + 4)

= 2x³ - 4x² + 8x + 3x² - 6x + 12

Step 2: Collect like terms

= 2x³ - x² + 2x + 12

Answer: 2x³ - x² + 2x + 12

Division of Polynomials

Example 5: Divide 2x³ - 3x² + 4x - 5 by x - 2

Result from long division:

Quotient = 2x² + x + 6

Remainder = 7

Final answer:

2x² + x + 6 + 7/(x - 2)

Answer: 2x² + x + 6 + 7/(x - 2)

Shortcut idea: If the question only asks for the remainder when dividing by (x - a), use the Remainder Theorem instead of long division.

4️⃣ Factor and Remainder Theorems

The Remainder Theorem

Remainder theorem
If P(x) is divided by (x - a), the remainder is P(a).

The Factor Theorem

Factor theorem
If P(a) = 0, then (x - a) is a factor of P(x).
Example 6: Find the remainder when P(x) = 2x³ - 5x² + 3x - 7 is divided by (x - 2)

Step 1: Use the Remainder Theorem

P(2) = 2(2³) - 5(2²) + 3(2) - 7

= 16 - 20 + 6 - 7

= -5

Answer: Remainder = -5
Example 7: Show that (x - 3) is a factor of x³ - 6x² + 11x - 6

Step 1: Evaluate at x = 3

P(3) = 3³ - 6(3²) + 11(3) - 6

= 27 - 54 + 33 - 6 = 0

Step 2: Apply the Factor Theorem

Since P(3) = 0, (x - 3) is a factor.

Answer: (x - 3) is a factor ✓

5️⃣ Factorization Techniques

A. Common Factor

Example 8: Factorize 6x³ + 9x² - 3x

Step 1: Take out the common factor

= 3x(2x² + 3x - 1)

Answer: 3x(2x² + 3x - 1)

B. Difference of Two Squares

Formula
a² - b² = (a + b)(a - b)
Example 9: Factorize 9x² - 16

= (3x)² - 4²

= (3x + 4)(3x - 4)

Answer: (3x + 4)(3x - 4)

C. Perfect Square Trinomials

Formulas
a² + 2ab + b² = (a + b)²
a² - 2ab + b² = (a - b)²
Example 10: Factorize 4x² - 12x + 9

= (2x)² - 2(2x)(3) + 3²

= (2x - 3)²

Answer: (2x - 3)²

D. Quadratic Trinomials

Example 11: Factorize 2x² + 7x + 3

Step 1: Multiply first and last coefficients

2 × 3 = 6

Step 2: Find two numbers that multiply to 6 and add to 7

6 and 1

Step 3: Split the middle term

= 2x² + 6x + x + 3

Step 4: Factor by grouping

= 2x(x + 3) + 1(x + 3)

= (2x + 1)(x + 3)

Answer: (2x + 1)(x + 3)

E. Sum and Difference of Cubes

Formulas
a³ + b³ = (a + b)(a² - ab + b²)
a³ - b³ = (a - b)(a² + ab + b²)
Example 12: Factorize 8x³ - 27

= (2x)³ - 3³

= (2x - 3)(4x² + 6x + 9)

Answer: (2x - 3)(4x² + 6x + 9)

F. Factorization by Grouping

Example 13: Factorize x³ + 2x² - x - 2

Step 1: Group the terms

= x²(x + 2) - 1(x + 2)

Step 2: Factor out the common bracket

= (x² - 1)(x + 2)

Step 3: Factor the difference of squares

= (x + 1)(x - 1)(x + 2)

Answer: (x + 1)(x - 1)(x + 2)

6️⃣ Roots of Polynomials

Finding Roots

Roots or zeros are the values of x for which P(x) = 0.

Example 14: Find the roots of x³ - 2x² - 5x + 6 = 0

Step 1: Test a simple factor

P(1) = 1 - 2 - 5 + 6 = 0

So (x - 1) is a factor.

Step 2: Divide to get the quadratic factor

x³ - 2x² - 5x + 6 = (x - 1)(x² - x - 6)

Step 3: Factor the quadratic

= (x - 1)(x - 3)(x + 2)

Answer: Roots are x = 1, 3, -2

Relationship Between Roots and Coefficients

Quadratic roots
For ax² + bx + c = 0 with roots α, β:
α + β = -b/a
αβ = c/a
Cubic roots
For ax³ + bx² + cx + d = 0 with roots α, β, γ:
α + β + γ = -b/a
αβ + αγ + βγ = c/a
αβγ = -d/a
Example 15: If α and β are roots of 2x² - 6x + 3 = 0, find α² + β²

Step 1: Use sum and product of roots

α + β = 6/2 = 3

αβ = 3/2

Step 2: Use identity

α² + β² = (α + β)² - 2αβ

= 9 - 2(3/2)

= 9 - 3 = 6

Answer: 6

7️⃣ Simultaneous Equations

One Linear and One Quadratic

The usual method is substitution. Replace the linear expression into the quadratic equation.

Example 16: Solve y = x + 1 and x² + y² = 13

Step 1: Substitute y = x + 1 into the second equation

x² + (x + 1)² = 13

Step 2: Expand

x² + x² + 2x + 1 = 13

2x² + 2x - 12 = 0

x² + x - 6 = 0

Step 3: Factorize

(x + 3)(x - 2) = 0

Step 4: Find corresponding y-values

If x = -3, then y = -2

If x = 2, then y = 3

Answer: (-3, -2) and (2, 3)

8️⃣ Graphs of Polynomials

Key Features to Identify

Feature Description
x-intercepts Where y = 0, that is, the roots
y-intercept Where x = 0, the constant term
Turning points Maximum and minimum points
Shape General direction and curve behavior

Example 17: Sketch y = x² - 4x + 3

Step 1: Find roots

(x - 3)(x - 1) = 0, so x = 1, 3

Step 2: Find y-intercept

At x = 0, y = 3

Step 3: Find turning point

x = -b/2a = 4/2 = 2

y = 2² - 4(2) + 3 = -1

Answer: Vertex is (2, -1)
Quadratic sketch idea
Mark the x-intercepts first, then the vertex, then sketch the curve smoothly.

Finding Maxima and Minima

Example 18: Find the turning points of y = x³ - 3x²

Step 1: Differentiate

dy/dx = 3x² - 6x

Step 2: Set derivative to zero

3x(x - 2) = 0

x = 0 or x = 2

Step 3: Find y-values

At x = 0, y = 0

At x = 2, y = 8 - 12 = -4

Answer: Turning points at (0, 0) and (2, -4)

🔟 Quick JAMB Tips

How to score high in polynomial questions

  • Always check your factors by expanding them back
  • Use the Remainder Theorem instead of long division when possible
  • For cubic equations, try simple factors of the constant term first
  • For graph questions, find intercepts before turning points
  • In simultaneous equations, substitute the linear equation into the quadratic
🧠 Quick strategy

When stuck, ask yourself: Is this a factorization question, a theorem question, or a graph question? That quickly narrows the method.

1️⃣1️⃣ Practice Questions

Q1: Find the remainder when 3x³ - 2x² + 4x - 5 is divided by (x - 2)

Step 1: Use the Remainder Theorem

P(2) = 3(8) - 2(4) + 4(2) - 5

= 24 - 8 + 8 - 5 = 19

Answer: 19
Q2: Factorize completely x³ - 3x² - 4x + 12

Step 1: Group terms

= x²(x - 3) - 4(x - 3)

Step 2: Factor common bracket

= (x² - 4)(x - 3)

Step 3: Factor difference of squares

= (x - 2)(x + 2)(x - 3)

Answer: (x - 3)(x - 2)(x + 2)
Q3: Solve simultaneously y = 2x - 1 and x² + y² = 5

Step 1: Substitute

x² + (2x - 1)² = 5

x² + 4x² - 4x + 1 = 5

5x² - 4x - 4 = 0

Step 2: Solve

x = [4 ± √96]/10 = [2 ± 2√6]/5

Step 3: Find corresponding y-values using y = 2x - 1

One exact solution is x = (2 - 2√6)/5, y = (-1 - 4√6)/5

The other is x = (2 + 2√6)/5, y = (-1 + 4√6)/5

Answer: The exact solutions are ((2 - 2√6)/5, (-1 - 4√6)/5) and ((2 + 2√6)/5, (-1 + 4√6)/5)
Q4: If (x - 2) is a factor of x³ + ax² - 5x + 2, find a

Step 1: Since (x - 2) is a factor, P(2)=0

2³ + a(2²) - 5(2) + 2 = 0

8 + 4a - 10 + 2 = 0

4a = 0

a = 0

Answer: a = 0
Q5: Find the minimum value of 2x² - 8x + 5

Step 1: Find the x-coordinate of the vertex

x = -b/2a = 8/4 = 2

Step 2: Substitute into the expression

2(2²) - 8(2) + 5 = 8 - 16 + 5 = -3

Answer: -3

1️⃣2️⃣ Quick Checklist for Polynomial Problems

🧠 Checklist

Check the degree first. Use the Remainder Theorem for quick remainder questions. Test factors using the Factor Theorem. For quadratics, check the discriminant when needed. Remember the sum and product of roots. For graphs, find x-intercepts, y-intercept, and the vertex. In simultaneous equations, substitute the linear into the quadratic.
