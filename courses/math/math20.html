📍 Measures of Location

JAMB Mathematics • mean • median • mode • quartiles • percentiles • grouped and ungrouped data

📋 Chapter Overview

This chapter covers mean, median, mode, quartiles, and percentiles. These are important statistical measures used to locate central or positional values in data sets.

You must be able to work with both ungrouped and grouped data.

  • Mean is the average value
  • Median is the middle value
  • Mode is the most frequent value
  • Quartiles divide data into 4 parts
  • Percentiles divide data into 100 parts
🧠 Big idea

Measures of location help you identify where the center or important positions of a data set lie.

1️⃣ Mean (Arithmetic Mean)

The Average

Formula
x̄ = ∑x / n
Where ∑x = sum of all values and n = number of values
Example 1: Simple Mean

Find the mean of: 5, 8, 12, 15, 20

x̄ = (5 + 8 + 12 + 15 + 20) / 5

= 60 / 5 = 12

Answer: Mean = 12
Example 2: Mean from Frequency Table
Score (x) Frequency (f) f × x
2 3 6
4 5 20
6 2 12
Total ∑f = 10 ∑fx = 38
← Swipe left/right to see more →

x̄ = ∑fx / ∑f = 38 / 10 = 3.8

Answer: Mean = 3.8

Easy memory tip: Mean = sum of everything ÷ how many things.

2️⃣ Median

The Middle Value

  • For odd number of values, take the middle value
  • For even number of values, take the average of the two middle values
Grouped data formula
Median = L + [(n/2 - Fb) / fm] × c
L = lower class boundary of median class
n = total frequency
Fb = cumulative frequency before median class
fm = frequency of median class
c = class width
Example 3: Median (Ungrouped - Odd)

Data: 3, 7, 8, 12, 15

Arranged data: 3, 7, 8, 12, 15

The middle value is the 3rd value.

Median = 8

Answer: Median = 8
Example 4: Median (Ungrouped - Even)

Data: 4, 6, 10, 14

The two middle values are 6 and 10.

Median = (6 + 10)/2 = 16/2 = 8

Answer: Median = 8
Example 5: Median from Grouped Data
Class Frequency (f) Cumulative Frequency
10-19 5 5
20-29 8 13
30-39 12 25
40-49 10 35
50-59 5 40
← Swipe left/right to see more →

Step 1: n = 40, so n/2 = 20

Step 2: Median class is 30-39

Step 3: L = 29.5, Fb = 13, fm = 12, c = 10

Step 4:

Median = 29.5 + [(20 - 13)/12] × 10

= 29.5 + (7/12) × 10

= 29.5 + 5.83 = 35.33

Answer: Median = 35.33

3️⃣ Mode

Most Frequent Value

For ungrouped data, the mode is simply the value that occurs most often.

Grouped data formula
Mode = L + [D1 / (D1 + D2)] × c
L = lower boundary of modal class
D1 = modal frequency − previous frequency
D2 = modal frequency − next frequency
c = class width
Example 6: Mode (Ungrouped)

Data: 2, 3, 3, 5, 7, 3, 8

2 appears 1 time

3 appears 3 times

5 appears 1 time

7 appears 1 time

8 appears 1 time

Answer: Mode = 3
Example 7: Mode from Grouped Data

Using the same table as Example 5:

Modal class is 30-39 because it has the highest frequency of 12.

L = 29.5

D1 = 12 - 8 = 4

D2 = 12 - 10 = 2

c = 10

Mode = 29.5 + [4 / (4 + 2)] × 10

= 29.5 + (4/6) × 10

= 29.5 + 6.67 = 36.17

Answer: Mode = 36.17

4️⃣ Percentiles and Quartiles

Data Division Points

  • Q₁ = 25th percentile
  • Q₂ = median = 50th percentile
  • Q₃ = 75th percentile
Quartile formula for grouped data
Qk = L + [(kn/4 - Fb) / f] × c
Where k = 1, 2, 3
Percentile formula
P = L + [(Pn/100 - Fb) / f] × c
Example 8: Finding Q₁

Using the grouped data from Example 5 where n = 40:

Step 1: Position of Q₁ = (1 × 40)/4 = 10

Step 2: Q₁ class is 20-29 because cumulative frequency 13 covers position 10

L = 19.5, Fb = 5, f = 8, c = 10

Q₁ = 19.5 + [(10 - 5)/8] × 10

= 19.5 + (5/8) × 10

= 19.5 + 6.25 = 25.75

Answer: Q₁ = 25.75

🎯 JAMB Exam Tips

How to avoid mistakes

  • Check whether the data is grouped or ungrouped before choosing a formula
  • Use class boundaries, not class limits, for grouped median and mode
  • Always calculate cumulative frequency first for median and quartiles
  • The modal class is the class with the highest frequency, not the final mode itself
  • For a symmetrical distribution, mean = median = mode
⚠️ Common mistakes

Do not use class limits when boundaries are required. Do not forget cumulative frequency. Do not confuse modal class with mode. Also be careful with quartile positions — use kn/4, not just n/4 for every quartile.
