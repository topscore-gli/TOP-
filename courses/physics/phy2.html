📚 Scalars & Vectors

JAMB Physics • classification • vector addition • components • displacement • position • units

🎯 Learning Objective

This chapter teaches you how to distinguish between scalar and vector quantities, resolve vectors into components, add vectors correctly, and understand position, distance, and displacement.

  • Scalars have magnitude only
  • Vectors have magnitude and direction
  • Distance is scalar
  • Displacement is vector
🧠 Easy memory rule

If direction matters, think vector. If only size matters, think scalar.

2️⃣ Vector Representation

Vectors can be shown in diagrams, symbols, and component form.

Graphical representation

  • A vector is drawn as an arrow
  • The length shows the magnitude
  • The arrowhead shows the direction

Mathematical representation

  • Arrow notation: F⃗, v⃗
  • Component form: (Fx, Fy)
  • Unit-vector form: Fxî + Fyĵ
Vector as an arrow
The arrow shows both size and direction. That is why vectors are different from scalars.

3️⃣ Vector Addition

When two or more vectors act together, we can combine them to get the resultant.

Method 1: Parallelogram Law

If two vectors act at a point, the diagonal of the parallelogram formed by them gives the resultant.

General resultant formula
R = √(A² + B² + 2ABcosθ)

Method 2: Triangle Law

Place the vectors head-to-tail. The resultant is drawn from the tail of the first to the head of the second.

Method 3: Component Method

This is the safest and most important method for JAMB.

Component method steps
Rx = Ax + Bx + ...
Ry = Ay + By + ...
R = √(Rx² + Ry²)
θ = tan⁻¹(Ry/Rx)
Vector addition by components
First resolve into horizontal and vertical parts, then combine them using Pythagoras.

5️⃣ Worked Examples on Vectors

Example 1: Two Perpendicular Forces

Two forces of 3 N and 4 N act at right angles. Find the resultant.

Step 1: Use R = √(3² + 4²)

Step 2: R = √(9 + 16) = √25 = 5 N

Step 3: Direction: tanθ = 4/3

Step 4: θ = 53.1°

Answer: 5 N at 53.1° to the 3 N force
Example 2: Vector Components

A displacement of 100 m makes 30° with the horizontal. Find its components.

Step 1: Horizontal = 100cos30° = 86.6 m

Step 2: Vertical = 100sin30° = 50 m

Answers: 86.6 m horizontally, 50 m vertically
Example 3: Multiple Vectors

A = 5 m at 0°, B = 3 m at 90°, C = 4 m at 180°

Step 1: Rₓ = 5 + 0 - 4 = 1

Step 2: Rᵧ = 0 + 3 + 0 = 3

Step 3: R = √(1² + 3²) = √10 = 3.16 m

Step 4: θ = tan⁻¹(3/1) = 71.6°

Answer: 3.16 m at 71.6° from the positive x-axis
Example 4: Relative Velocity

A boat moves north at 5 m/s across a river flowing east at 3 m/s. Find the resultant velocity.

Step 1: Represent as 3î + 5ĵ

Step 2: Magnitude = √(3² + 5²) = √34 = 5.83 m/s

Step 3: Direction = tan⁻¹(5/3) = 59.0°

Answer: 5.83 m/s at 59.0° north of east

6️⃣ Measurement, Position, Distance and Displacement

Position

Position tells where an object is relative to a reference point.

Distance

Distance is the total path length travelled. It is a scalar.

Displacement

Displacement is the straight-line change in position from start to finish. It is a vector.

Concept Definition Nature Unit
Position Location relative to origin Vector m
Distance Total path length Scalar m
Displacement Change in position Vector m
← Swipe table to see more →
Distance vs displacement
Distance follows the actual path. Displacement is the straight line from start to finish.

7️⃣ Coordinate Systems

1-Dimension

Position is given by x. Displacement is:

1D displacement
Δx = xf - xi

2-Dimension

Position is given by coordinates (x, y).

2D displacement
Δr⃗ = (xf - xi)î + (yf - yi)ĵ
Magnitude = √((Δx)² + (Δy)²)
Position on x-y plane
Displacement in 2D is found by subtracting the coordinates of the start point from the finish point.

8️⃣ Worked Examples on Distance and Displacement

Example 5: Distance vs Displacement

A student walks 400 m east, 300 m north, 400 m west, and 300 m south.

Step 1: Distance = 400 + 300 + 400 + 300 = 1400 m

Step 2: East-West displacement = 400 - 400 = 0

Step 3: North-South displacement = 300 - 300 = 0

Answers: Distance = 1400 m, Displacement = 0 m
Example 6: Displacement in 2D

A particle moves from A(2,3) to B(5,7).

Step 1: Δx = 5 - 2 = 3

Step 2: Δy = 7 - 3 = 4

Step 3: Vector = 3î + 4ĵ

Step 4: Magnitude = √(3² + 4²) = 5 m

Step 5: Direction = tan⁻¹(4/3) = 53.1°

Answer: 5 m at 53.1°
Example 7: Average Speed vs Average Velocity

A car travels 60 km north in 1 hour, then 80 km east in 2 hours.

Step 1: Total distance = 60 + 80 = 140 km

Step 2: Total time = 3 h

Step 3: Average speed = 140/3 = 46.7 km/h

Step 4: Displacement = √(60² + 80²) = 100 km

Step 5: Average velocity = 100/3 = 33.3 km/h

Answers: 46.7 km/h, and 33.3 km/h at 53.1° east of north

Convert 72 km/h to m/s.

1️⃣1️⃣ JAMB Exam Tips

How to avoid common mistakes

  • Always check if direction is required
  • If it is a vector, magnitude alone is incomplete
  • Draw a sketch whenever possible
  • Distance is total path, displacement is shortest straight-line path
  • For 3 or more vectors, use component method
  • Watch for negative signs in leftward and downward directions
