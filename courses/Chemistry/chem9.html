🧠 One-liner

Oxidation = loss e⁻ = ON up • Reduction = gain e⁻ = ON down.

1️⃣ Classical Definitions (Oxygen Method)

Oxidation is the addition of oxygen to a substance.

Example: 2Mg(s) + O₂(g) → 2MgO(s)

Reduction is the removal of oxygen from a compound (often by hydrogen).

Example: CuO(s) + H₂(g) → Cu(s) + H₂O(l)

Classical view: oxidation adds oxygen; reduction removes oxygen

2️⃣ Modern Definition (Electrons) — OIL RIG

Oxidation = loss of electrons (OIL).

Reduction = gain of electrons (RIG).

Redox happens by electron transfer (half-reactions)

Step-by-step example: 2Na + Cl₂ → 2NaCl

  • Oxidation half-reaction: Na → Na⁺ + e⁻
  • Reduction half-reaction: Cl₂ + 2e⁻ → 2Cl⁻
  • Balance electrons: multiply Na half by 2
  • Combine: 2Na + Cl₂ → 2NaCl

Another quick example: 2Mg + O₂ → 2MgO

Mg: 0 → +2 (loss of 2e⁻) → oxidised

O: 0 → −2 (gain of 2e⁻) → reduced

3️⃣ Oxidation Numbers (ON) + Changes

Oxidation = increase in ON. Reduction = decrease in ON.

Oxidation number rules (6 rules)

  • Free elements = 0
  • Group 1 = +1, Group 2 = +2
  • H = +1 (except hydrides = −1)
  • O = −2 (except peroxides = −1)
  • Sum of ON in neutral compound = 0
  • Sum of ON in an ion = charge on the ion

Example (very common)

  • In KMnO₄, Mn = +7

4️⃣ Balancing Redox Using Oxidation Numbers

Ababio method: assign ON → note total change → balance e⁻ lost/gained → balance remaining atoms.

Worked example

Unbalanced: MnO₂ + HCl → MnCl₂ + Cl₂ + H₂O

Balanced: MnO₂ + 4HCl → MnCl₂ + Cl₂ + 2H₂O

🧠 Balancing tip

Start by balancing the atoms that change ON (then H and O last).

5️⃣ IUPAC Nomenclature Using Oxidation Number

Find the oxidation state first, then name with Roman numerals and oxo- prefixes.

📝 6 JAMB Examples (Step-by-Step)

Example 1: Classical (oxygen)

CuO + H₂ → Cu + H₂O: which is oxidised?

CuO loses oxygen → reduced

H₂ gains oxygen → oxidised

Answer: H₂ is oxidised

Example 2: Electron transfer

Show electron transfer in 2Na + Cl₂ → 2NaCl

Na → Na⁺ + e⁻ (oxidation)

Cl₂ + 2e⁻ → 2Cl⁻ (reduction)

Example 3: ON change

H₂S + Cl₂ → 2HCl + S: which element is oxidised?

S in H₂S: −2 → 0 (increase) → oxidised

Answer: Sulphur is oxidised

Example 4: Balance using ON method

Balance: MnO₂ + HCl → MnCl₂ + Cl₂ + H₂O

Balanced: MnO₂ + 4HCl → MnCl₂ + Cl₂ + 2H₂O

Example 5: IUPAC naming

Give IUPAC name for K₂Cr₂O₇

Cr oxidation number = +6

Answer: Potassium heptaoxodichromate(VI)

Example 6: Identify oxidising agent

In 2KMnO₄ + 16HCl → products, which is oxidising agent?

Mn: +7 → +2 (decrease) → KMnO₄ is reduced

Oxidising agent is the substance reduced

Answer: KMnO₄ is the oxidising agent
